# CSS DISPLAY PROPERTY

## Has 4 different values

- block
- inline 
- inline-block
- none 

### COMMON BLOCK ELEMENTS
- Paragraph `<p> </p>`
- Header `<h1> through <h6>`
- Division `<div>`
- List and List item `<ol>,<ul>,<li>`
- Form `<form> </form>`
- Footer `<footer> </footer>`
- Section `<section> </section>`

### COMMON INLILNE ELEMENTS
- Spans `<span> </span>`
- Images `<img>`
- Anchors `<a> </a>`
